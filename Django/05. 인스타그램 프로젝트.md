# 인스타그램 프로젝트

1. 프로젝트로 사용할 폴더 생성
2. pyenv virtualenv 3.4.3 <환경명>
3. pyenv local <환경명>
4. pip install django
5. django-admin startproject <프로젝트명>
6. mv <프로젝트명> django_app
7. Pycharm interpreter세팅
8. `django_app`폴더를 `Sources root`로 설정
